# GraphRAG Chatbot web UI

Optional Vite + React chat front end for the GraphRAG Chatbot API (`POST /prompt`). It is intended for local development and demos only—do not deploy or rely on it in production.

Copy `.env.example` to `.env` and set `VITE_API_BASE_URL` to your API base URL (no trailing slash). Run `npm install` and `npm run dev` from this directory with the backend reachable at that URL.
